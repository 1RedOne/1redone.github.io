## Introducing Quick Web Tools

I just shipped a new tool: **[Quick Web Tools](https://www.foxdeploy.com/QuickWebTools)**.

### What it does

I always need to make minor text conversions to help with queries, parsing data, you name it.  So I decided to just make a tool to help me do this more easily!

Right now, it will convert a list of items into a ready to use Kusto `in ("abc","cde","xyz")` style query component.  Tomorrow...who knows!

### How to use it

- Open **[Quick Web Tools](https://www.foxdeploy.com/QuickWebTools)**

### Notes

As always, no analytics, no saving, no copying.  All conversions happen locally and nothing leaves your box.
